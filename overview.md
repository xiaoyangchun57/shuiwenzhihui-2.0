# 平台端主题色切换功能完成

## 行业配色方案（咨询总结）

水利水文行业平台常见色调：
- **主流选择**：蓝白基调（水蓝色配白色背景），如数字孪生流域、水文监测系统
- **技术路线**：浅色主题为主流，深色主题较少见但科技感强
- **本次方案**：保留现有深色（青绿科技风）为默认，新增浅色（白底水蓝字）

## 技术实现

### 方案：CSS 自定义属性 + data-theme 属性切换

| 项目 | 方案 |
|------|------|
| 技术选型 | 46 个 CSS 变量（`:root` 定义），`[data-theme="light"]` 覆盖 |
| 切换方式 | 导航栏月亮/太阳图标按钮，点击即时切换 |
| 持久化 | `localStorage` 保存偏好，刷新后自动恢复 |
| 覆盖范围 | dashboard.css 全部 330+ 行颜色已替换为变量引用 |

### 浅色主题配色

```css
--bg-body: #f0f6fb      /* 浅蓝灰背景 */
--bg-card: #ffffff       /* 纯白卡片 */
--text-primary: #1a334a  /* 深蓝字 */
--text-secondary: #4a8aaa /* 水蓝色 */
--text-accent: #00b8a0   /* 青色点缀 */
```

### 注意事项

| 关注点 | 处理方式 |
|--------|---------|
| 状态色（红/橙/黄/绿） | 微调为浅色主题下更柔和的版本 |
| Leaflet 地图弹窗 | 使用 `var()` 变量自动适配 |
| Chart.js 图表 | 保留深色底色，只在浅色主题下微调 |
| 地图底图（高德卫星图） | 保持不变（卫星图不依赖主题） |
| MarkerCluster 聚合图标 | 保持默认（半透明叠加在图上） |
| 内联脚本体积 | 新增约 0.5KB，远低于 210K 限制 |

## 使用方式

导航栏时钟右侧新增 🌙/☀️ 切换按钮（SVG 图标），点击即可切换主题，刷新后自动记住选择。
