# 水文监测智慧运营平台

水利监测站点综合管理、告警处置、工单流转、运维巡检一体化平台。

## 功能模块

- **信息中心（驾驶舱）**：地图展示、实时数据、告警概览、数据到报率
- **站点管理**：267+ 监测站点统一管理
- **预警中心**：告警分级、督办、转工单、批量处理
- **工单管理**：工单全生命周期流转（待受理→已受理→...→已关闭）
- **运维管理**：巡检计划、运维计划、进度追踪、审核
- **水位校验**：人工/遥测水位差值校验、异常转工单

## 技术栈

- **前端**：纯 HTML/CSS/JS（单页面应用）
- **地图**：Leaflet + MarkerCluster + 高德卫星图
- **图表**：Chart.js
- **后端**：Flask (Python)
- **数据库**：SQLite

## 快速启动

```bash
cd backend
python app.py
```

访问 http://localhost:5000
